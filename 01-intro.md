# Programming Bitcoin on Urbit

## Intro
Goal: to be confident reading, writing and manipulating 
data structures on the Bitcoin blockchain, 
using Urbit/Hoon in particular.

## Topics Covered
* ECDSA
* bip32 wallets
* transactions/utxos
* tx signing
* spending scripts
* bitcoin on Urbit
* taproot/Schnorr

## Topics Not Covered
* Lightning (next workshop)
